# Documentation

这是我的[个人文档](https://nekoside.com/docs/)

作为[静态页面](https://www.halo.run/store/apps/app-gFkMn)部署在 [Halo](https://github.com/halo-dev/halo) 博客站点下

# Installation

```bash
pnpm install
```

# Preview

```bash
pnpm dev
```

# Build

```bash
pnpm build
```

# Deploy

本项目使用 [halo-static-pages-deploy-cli](https://www.npmjs.com/package/halo-static-pages-deploy-cli) 进行部署

填写 `_settings.toml` 后重命名为 `settings.toml` 再运行 `deploy.py` 即可.
