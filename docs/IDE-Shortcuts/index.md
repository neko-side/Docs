# IDE-Shortcuts

## 打开终端
- Windows
    - `Ctrl + ~`
- macOS
    - `Cmd + ~`

## 打开设置
- Windows
    - `Ctrl + ,`
- macOS
    - `Cmd + ,`

## 快捷键设置

- Windows
  - `Ctrl + Alt + S`
- macOS
  - `Cmd + Alt + S`

## 运行

- Windows
  - `F5`
- macOS
  - `F5`

## Debug

- Windows
  - `Ctrl + F5`
- macOS
  - `Cmd + F5`

## Build

- Windows
  - `F7`
- macOS
  - `F7`

## 切换 Tab

- Windows
  - `Ctrl + Num`
- macOS
  - `Cmd + Num`
